Number Systems

Number systems is mathematical framework that uses a set of symbols and rules to represent quantities, allowing for counting, measuring, and calculation


There are four types of Number systems

  • Binary
  • Octal
  • Decimal
  • Hexadecimal

    • Significance and Origin

  • Binary
    • Computer Language - 1's and 0's
  • Octal
    • Used as a shorthand for binary. Most famous during the rise of early computers
  • Decimal
    • Human Friendly System
  • Hexadecimal
    • Human Programming Language for programmers and more compact than binary
